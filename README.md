# SE302 OLX Testing Project

Automated tests for [olx.ba](https://olx.ba/) using Playwright.

## Project Structure

- `page-objects/`: Contains Page Object Model classes.
- `tests/`: Contains test specifications.
- `playwright.config.ts`: Playwright configuration.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Install Playwright browsers:
   ```bash
   npx playwright install
   ```

## Running Tests

Run all tests:
```bash
npx playwright test
```
